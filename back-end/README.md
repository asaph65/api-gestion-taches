# My Task API - API de Gestion de Tâches

API RESTful pour la gestion de tâches, construite avec Node.js, Express et MongoDB.

## Installation et Démarrage

### Prérequis

- Node.js
- MongoDB compass
- npm

### Installation

1. **Cloner le projet**
   ```bash
   git clone <repository-url>
   cd my-task-api
   ```
